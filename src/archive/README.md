# Archive — Superseded Work

Scripts and notes from earlier iterations of this repository, kept for provenance. Not part of the current portfolio.

## Contents

- **`README_original.md`** — the original repo README (pre-portfolio, when the work was framed as "AI Pathway Economics" single-study).
- **`poc_*.py`** — early proof-of-concept scripts (econometric audit, life-stage perturbation, mitigation economy, model collapse, triage preference). These informed the methodology that became the current portfolio.
- **`run_*.py`** — earlier study-orchestration scripts (experiment, full-study, pathway-study, study). Superseded by the `src/pilot_*.py` + `src/analyze_*.py` scripts.
- **`analyze.py`, `bias_stress_test.py`, `eu_ai_act_audit.py`, `parse_responses.py`, `score.py`, `vignette_builder.py`, `visualize.py`** — legacy utility scripts.
- **`notebooks/`** — exploratory notebook.
- **`pathway_comparison_report.md`** — output from an early pathway comparison study.

None of this is referenced by the current portfolio. Kept to show the development trajectory and because some of the parametrization (cascade model, published cost parameters) originated here.
